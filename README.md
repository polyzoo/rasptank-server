# RaspTank Server

Сервер управления машинкой на Raspberry Pi: автономное движение, контроль препятствий, REST API.

## Требования

- Python 3.7 или выше.
- Raspberry Pi с подключёнными моторами и датчиком HC-SR04.

## Быстрый старт

1. Клонируйте репозиторий и перейдите в каталог проекта.

```bash
git clone git@github.com:polyzoo/rasptank-server.git
cd rasptank-server
```

2. Создайте и активируйте виртуальное окружение.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Установите зависимости.

```bash
pip install -r requirements.txt
```

4. Запустите веб-сервер.

```bash
uvicorn src.main:app --host 0.0.0.0 --port 8010
```

5. Перейдите по адресу `http://localhost:8010/docs`, если все открывается, значит веб-сервер
   работает корректно.

## Запуск тестового скрипта

Скрипт дублирует API-команды через командную строку — удобно для проверки моторов и датчика без
FastAPI.

```bash
python3 test_drive.py <команда> [аргументы]
```

| Команда                                | Описание                                                         |
|----------------------------------------|------------------------------------------------------------------|
| `forward <расстояние_см> [скорость_%]` | Движение вперёд на заданное расстояние                           |
| `stop [--delay N]`                     | Запуск движения, пауза N секунд, остановка (для проверки `stop`) |

### Примеры запуска

Движение вперёд на 100 см (скорость по умолчанию).

```bash
python3 test_drive.py forward 100
```

Движение на 50 см с максимальной скоростью 60%.

```bash
python3 test_drive.py forward 50 60
```

Проверка остановки (движение ~2 с, затем остановка).

```bash
python3 test_drive.py stop
```

Проверка остановки с паузой 3 секунды.

```bash
python3 test_drive.py stop --delay 3
```
